# Memoir Workspace

This folder contains the canonical memoir structure for **Will I Bring the Chaos**.

Notion remains the active writing studio for drafts, fragments, character notes, and evolving scenes.

GitHub holds canonical exports, arc definitions, voice rules, and completed or stabilized memoir artifacts.

## Rule

A memoir piece belongs in GitHub when it has become durable enough to serve as reference material for future writing, editing, agent behavior, or structural planning.

## Current Canonical Sources

- `identities/MEMOIR_PERSONALITY.md` — narrator identity and memoir lens
- `memoir/arcs/README.md` — memoir arc map
- `memoir/canon/README.md` — stabilized story exports

## Working Thesis

The memoir is not a confession machine.

It is a pattern-recognition engine with jokes, scars, bad decisions, loyalty, absurdity, and a stubborn belief that people are rarely reducible to the worst thing they ever did.
