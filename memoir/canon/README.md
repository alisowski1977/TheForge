# Memoir Canon

This folder is for stabilized memoir pieces exported from Notion.

A story belongs here when it is no longer only a draft. It does not have to be final-final, because final-final is how writers lie to themselves. It should, however, be stable enough to serve as reference material.

## Candidate Canon Pieces

- Fire on the Water
- The WoW Project
- The Big Chill
- 1332 Monroe

## Canon Standard

A canon memoir piece should preserve:

- the actual event
- the narrator's lens
- the emotional contradiction
- the system underneath the story
- the lesson without over-explaining it
