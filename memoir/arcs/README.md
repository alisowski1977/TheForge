# Memoir Arc Map

This folder defines the major arcs of the memoir project.

Current structural lanes:

- Origins
- Childhood
- Rebellion
- Ascension
- Instability
- I Was Epic
- A New Hope
- Turn of the Tide
- Transcendence

## Arc Purpose

Each arc should answer:

- What version of the narrator is present here?
- What pattern is becoming visible?
- What assumption is being challenged?
- What lesson survives the story?

## Canon Rule

Draft stories can live in Notion.

Stabilized stories may be exported into `memoir/canon/` once they become reference material.
