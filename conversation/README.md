# Conversation

This folder is for raw or lightly summarized conversational material that has not yet earned promotion.

Conversation is the lowest-pressure layer of The Forge.

It may include:

- session notes
- rough summaries
- unresolved questions
- interesting fragments
- possible future artifacts
- material that should not be lost but is not yet doctrine

## Rule

Conversation is not canon.

A conversation may contain the seed of canon, but the seed still has to grow teeth.
