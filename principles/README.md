# Principles

This folder is for reusable models that have survived enough challenge to guide future reasoning.

A principle says:

> This has been tested enough to use again.

## Standard

A principle should be:

- clear
- reusable
- domain-aware
- falsifiable enough to critique
- useful outside the original conversation
- grounded in experience rather than vibes

## Governance

The Clerk may recommend promotion to principle.

Andy approves promotion.

Approval should be explicit.

## Rule

A principle is strong, but it is not automatically canon.

Promotion path:

```text
Principle → Canon
```
