# Observations

This folder is for noticed patterns that are worth preserving but are not yet full explanations or principles.

An observation says:

> This happened often enough, clearly enough, or strangely enough that it should not be lost.

## Examples

An observation may come from:

- a memoir scene
- an ERP incident
- a leadership moment
- a repeated organizational failure
- a relationship pattern
- a governance problem
- a useful phrase that points at something real

## Rule

Observations should be specific.

They should not pretend to be universal laws.

Promotion path:

```text
Observation → Hypothesis
```
