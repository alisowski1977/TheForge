# Hypotheses

This folder is for proposed explanations that have not yet earned principle status.

A hypothesis says:

> I think this pattern means something.

## Standard

A hypothesis should be clear enough to challenge.

It should identify:

- the observed pattern
- the proposed explanation
- where it might apply
- what would weaken or disprove it
- whether it belongs to memoir, systems, leadership, governance, or another domain

## Rule

A hypothesis is not yet a principle.

It may be useful.
It may be persuasive.
It may even be mostly right.

But it still needs to survive critique.

Promotion path:

```text
Hypothesis → Principle
```
